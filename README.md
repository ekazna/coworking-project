**Дипломная работа "Разработка сервиса автоматизации базовых процессов управления коворкингом"**

Backend: Python, Django, DRF

Frontend: React

DB: PostgreSQL

Интеграции: Telegram Bot API


Разработаны:
 - личный кабинет
 - редактирование профиля
 - страница бронирования, продления бронирования
 - просмотр активных и прошлых броней
 - подача обращения
 - интерфейсы администратора:
   - брони
   - управление поломками
   - управление ресурсами
   - управление клиентами
   - аналитика

Ключевые сущности: 
- Пользователи (User, Profile)
- Ресурсы (Category, Type, Resource)
- Бронирования (Booking, ChangeLog)
- Обращения (Issue, ResourceOutage)
- Уведомления (Notification)

Особенности: 
- поддержка связанных броней
- хранение истории изменений
- учёт недоступности ресурсов
